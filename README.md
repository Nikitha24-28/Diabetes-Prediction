# Diabetes-Prediction

Diabetes Prediction System
This project is a Machine Learning model that predicts whether a person is likely to have diabetes based on medical parameters.
I built this project to understand the complete ML workflow — from data preprocessing to model training and real-time prediction.

Dataset
The dataset includes features like:
Pregnancies
Glucose
Blood Pressure
Insulin
BMI
Age

Target:
0 → Non-Diabetic
1 → Diabetic

Tech Stack:                             
Python
NumPy
Pandas
Scikit-learn

Key Steps :                          
Data preprocessing & feature scaling                                      
Train-test split                                                
Model training (Logistic Regression / SVM)                                         
Accuracy evaluation                                                       

Result
The model successfully predicts whether a person is diabetic based on input medical data.

Built as part of my journey into Machine Learning and real-world predictive systems
